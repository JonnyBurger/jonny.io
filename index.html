Komplexe Web-Apps, made in Switzerland.

Ich erstelle interaktive Websites, die von Menschen aus der ganzen Welt benutzt werden. Beispiel gefällig? Schauen sie auf onetune.fm vorbei und geniessen sie ein paar Songs.

Front- und Backend aus einer Hand.

Jede Applikation besteht aus einem Interface, wie zum Beispiel einer Website, und einem Server zur Kommunikation. Ich mache beides - Frontends erstellt mit HTML und CSS und Backends mit node.js.

Simples und funktionales Design, selbstgemacht.

Eine einzigartige Identität kann nicht aus Templates entstehen. Deshalb wird bis auf den letzten Pixel alles selber gestaltet, um der Website ein einheitliches Interface zu geben.
